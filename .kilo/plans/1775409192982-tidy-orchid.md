### 2. Step-by-step Data Flow

1.  **Инициализация:** Создание папки `lesson_24/`, файла логов `app_24.log` и базы данных `parabola.db`.
2.  **Запуск UI:** Скрипт `run_lesson_24.py` строит интерфейс Gradio с двумя колонками.
3.  **Генерация данных (Action: Generate Data):**
    *   Пользователь вводит `a`, `c`, `x_min`, `x_max`.
    *   `ui_controller` вызывает `config_manager.save_config`.
    *   `logic.calculate_parabola` генерирует список кортежей $(x, y)$.
    *   `db_manager.save_points` очищает старую таблицу и записывает новые точки.
    *   `db_manager.get_points` возвращает данные для отображения в `gr.Dataframe`.
4.  **Визуализация (Action: Draw Graph):**
    *   `ui_controller` вызывает `db_manager.get_points`.
    *   Данные преобразуются в `pandas.DataFrame`.
    *   Создается объект `plotly.graph_objects.Figure` с кривой параболы.
    *   Объект возвращается в `gr.Plot`.
