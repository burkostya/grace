### 2. Step-by-step Data Flow (Пошаговый Поток Данных)

1.  **Инициализация:** При запуске `run_lesson_23.py` инициализируется БД SQLite (`lesson_23/parabola_23.db`) и создается интерфейс Gradio.
2.  **Ввод параметров:** Пользователь в левой колонке UI выставляет значения `a`, `c`, `x_min`, `x_max`.
3.  **Генерация данных (Кнопка 1):**
    -   Вызывается `handle_generate`.
    -   `ConfigManager` сохраняет параметры в `lesson_23/config.json`.
    -   `logic.generate_parabola_points` рассчитывает точки $y = ax^2 + c$ для заданного диапазона.
    -   `DBManager` сохраняет точки в таблицу `points` базы данных SQLite.
    -   UI считывает данные из БД и обновляет компонент `Dataframe`.
4.  **Построение графика (Кнопка 2):**
    -   Вызывается `handle_draw`.
    -   `DBManager` извлекает все точки из БД.
    -   Создается объект `plotly.graph_objects.Figure` с кривой параболы.
    -   UI обновляет компонент `Plot` в правой колонке.
5.  **Логирование:** Каждый этап сопровождается LDD-логами в `lesson_23/app_23.log` с указанием `[IMP:1-10]` и `Belief State`.
